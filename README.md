# Kardes – AI Crypto Automation Bot

Kardes is an AI-powered crypto automation bot designed to monitor cryptocurrency markets and generate trading insights.

Features:

• Binance Top Gainers Monitoring  
• Trade Signal Generation (3–5%)  
• Technical Analysis  
• Binance Blog Monitoring  
• Crypto News Updates  
• WhatsApp Alert System  
• Binance Square Auto Posting  
• CreatorPad Task Automation  
• Smart Contract Assistance  
• Binance Latest campaigns announcement update 
• Weather update.
• Updates from X accounts like @saylor and @cz_Binance


How it works:

1. The bot scans Binance top gaining coins.
2. AI analyzes market momentum.
3. Trade signals are generated.
4. Signals are posted on Binance Square.
5. Alerts are sent to WhatsApp.

Purpose:

The goal of Kardes is to automate crypto market research and trading analysis while maintaining active engagement on Binance Square.
